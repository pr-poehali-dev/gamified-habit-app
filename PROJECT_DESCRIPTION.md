# СтарКидс — Описание проекта

> tasks4kids.ru · Telegram Mini App · React + Python · May 2026

---

## Что такое СтарКидс

**СтарКидс** — это геймифицированная система заданий и наград для детей и родителей. Приложение превращает домашние обязанности, учёбу и полезные привычки в игру: ребёнок выполняет задания, зарабатывает звёзды и обменивает их на реальные призы, которые придумал родитель.

Работает как **Telegram Mini App** (через ботов) и как **PWA** (прогрессивное веб-приложение) для тех, кто не использует Telegram. Два отдельных бота: `@parenttask_bot` для родителей и `@task4kids_bot` для детей.

---

## Какие задачи решает

| Проблема | Решение |
|----------|---------|
| Дети не хотят делать домашние дела | Задания оформлены как квесты с наградой в звёздах |
| Нет системы контроля выполнения | Родитель подтверждает каждое задание (с фото или без) |
| Сложно мотивировать через слова | Звёзды → реальные призы (выбирает сам родитель) |
| Дети не следят за учёбой | Ребёнок сдаёт оценки через приложение, получает звёзды |
| Родитель не знает, что происходит | Telegram-уведомления о каждом действии ребёнка |
| Нет системы прогресса у ребёнка | Уровни, достижения, стрики, рейтинг среди друзей |

## Аудитория

- **Основная:** Родители детей 6–16 лет, которые хотят приучить ребёнка к ответственности
- **Вторичная:** Сами дети как активные пользователи детского бота
- **Рынок:** Русскоязычные семьи, использующие Telegram

## Структура приложения

Приложение состоит из двух независимых интерфейсов — для родителя и для ребёнка.

### Родительский интерфейс (5 вкладок)

#### 1. Задания
- Создание задания: название, эмодзи, награда (звёзды), дедлайн
- Опция «требовать фото» для подтверждения выполнения (Premium)
- Подтверждение или отклонение выполненных заданий
- Обработка запросов на продление дедлайна
- Статусы: активно / ожидает подтверждения / выполнено / отклонено

#### 2. Оценки
- Просмотр оценок, которые ребёнок сдал через своё приложение
- Подтверждение или отклонение оценки
- При подтверждении — автоматическое начисление звёзд (5→5⭐, 4→4⭐, 3→3⭐)
- История всех оценок с датами

#### 3. Бонусы (награды)
- Создание пула наград для каждого ребёнка (например: мороженое = 10⭐, новая игра = 50⭐)
- Просмотр запросов на покупку награды от ребёнка
- Список желаний ребёнка — что он хочет получить в подарок

#### 4. Ребёнок / Профиль
- Добавление детей по инвайт-коду
- Статистика каждого ребёнка: уровень, звёзды, достижения
- Управление профилями детей

#### 5. Профиль родителя
- Уровень и XP родителя (система прогресса)
- Стрик активности (серия дней подряд)
- Настройки уведомлений
- Управление подпиской Premium

---

### Детский интерфейс (5 вкладок)

#### 1. Задания
- Список активных заданий с дедлайнами и наградой
- Кнопка «Выполнено» с опциональной загрузкой фото
- Запрос продления, если не успевает
- Начисление звёзд после подтверждения родителем

#### 2. Оценки (доступно с 2 уровня)
- Ввод предмета и оценки
- Автоматический расчёт звёзд за хорошую оценку
- Список предметов подбирается по возрасту ребёнка (6–18 лет)

#### 3. Магазин наград
- Каталог наград, созданных родителем
- Покупка за звёзды
- История покупок и их статус
- Добавление товаров в список желаний

#### 4. Друзья
- Добавление друзей по уникальному коду
- Рейтинг друзей по количеству звёзд
- Обработка входящих/исходящих заявок

#### 5. Профиль ребёнка
- Текущий уровень и прогресс до следующего (10⭐ = 1 уровень)
- Коллекция достижений (16 штук)
- Коллекция стикеров
- Инвайт-код для подключения к родителю

## Система прогресса

### Прогресс ребёнка

| Уровень | Название | Значок |
|---------|----------|--------|
| 1–2 | Новичок | ⭐ |
| 3–4 | Бронза | 🥉 |
| 5–6 | Серебро | 🥈 |
| 7–8 | Золото | 🥇 |
| 9+ | Алмаз | 💎 |
| 10+ | Чемпион | 🏆 |

10 звёзд = 1 уровень. Звёзды копятся общей суммой.

### Достижения ребёнка (16 штук)

| Категория | Достижения |
|-----------|-----------|
| Задания | Первое задание, 5 заданий, 10 заданий, 25 заданий |
| Звёзды | 10 звёзд, 50 звёзд, 100 звёзд |
| Уровни | Уровень 3, уровень 5, уровень 10 |
| Траты | Потратить 10⭐, потратить 30⭐ |
| Награды | Первая покупка, 3 покупки |
| Стрики | 3 дня подряд, 7 дней подряд |

### Прогресс родителя

Родитель тоже прокачивается: получает XP и очки за подтверждение заданий, оценок, ежедневную активность. Стрик бонусирует за ежедневное использование. Уровни: Начинающий → Активный → Опытный → Продвинутый → Мастер.

## Технические разделы

### Авторизация
Два способа входа:
1. **Через Telegram** — автоматически по initData (HMAC-SHA256 верификация)
2. **Через PWA** — телефон + OTP-код (SMS через SMSAero) → PIN-код на будущее

### Уведомления
- **Telegram-уведомления** родителю о каждом действии ребёнка
- **Push-уведомления** через Web Push API (Service Worker)
- Автоматические триггеры (нет детей 24ч, неактивен 3 дня, пробный заканчивается)
- Тихие часы: 20:00–09:00 по Москве

### Фото-подтверждение
- Ребёнок делает фото выполненного задания
- Фото загружается в S3 (CDN: cdn.poehali.dev)
- Родитель видит фото при подтверждении
- Только для Premium-подписчиков

### Оценки из школы
Предметы подбираются динамически по возрасту ребёнка:
- 6–7 лет: математика, чтение, рисование, физкультура
- 8–10 лет: + русский, окружающий мир, труд
- 11–14 лет: + алгебра, геометрия, история, биология, физика, химия
- 15–18 лет: + обществознание, информатика, иностранный язык

## Идеи для улучшений и доработок

### Высокий приоритет

1. **Семейный чат / комментарии к заданиям** — родитель может оставить голосовое или текстовое сообщение к заданию, ребёнок отвечает

2. **Еженедельный отчёт родителю** — автоматическое PDF/Telegram-сообщение с итогами недели: сколько заданий выполнено, какие оценки, сколько звёзд заработано

3. **Шаблоны заданий** — готовые наборы заданий по возрасту (убрать в комнате, почистить зубы, прочитать 20 страниц). Быстрый старт для новых родителей

4. **Групповые задания** — одно задание для нескольких детей сразу (например, вместе убрать квартиру)

5. **Повторяющиеся задания** — ежедневные/еженедельные задачи без ручного создания (почистить зубы = ежедневно)

### Средний приоритет

6. **Рейтинг достижений в семье** — если несколько детей, сравнение между ними в виде красивой таблицы

7. **Маркетплейс наград** — готовые идеи наград (кино, пицца, игровое время), которые родитель может быстро добавить

8. **Партнёрские магазины** — интеграция с реальными магазинами: ребёнок обменивает звёзды на купоны (Буквоед, Читай-город и т.д.)

9. **Напоминания для детей** — уведомление ребёнку о дедлайне задания за 2 часа до истечения

10. **Дни особых заданий** — праздничные квесты на Новый год, День рождения, День знаний с увеличенными наградами

### Долгосрочные

11. **Веб-версия для родителей** — полноценный личный кабинет в браузере (не только Telegram Mini App)

12. **Интеграция с дневником Госуслуг / Сферум** — автоматическое получение оценок без ручного ввода

13. **Виртуальный питомец** — ребёнок кормит/ухаживает за питомцем на звёзды, питомец растёт вместе с уровнем

14. **Режим «Копилка»** — ребёнок копит звёзды на большую цель с визуальным прогресс-баром

15. **Родительский контроль экранного времени** — интеграция задания «сделал уроки» с разблокировкой времени в телефоне

16. **Статистика для родителя по месяцам** — графики: активность, рост уровня, выполнение в % от заданных

17. **AI-помощник по заданиям** — генерация идей заданий на основе возраста и интересов ребёнка
